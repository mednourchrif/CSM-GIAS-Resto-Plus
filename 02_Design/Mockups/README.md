# Dossier Mockups — Maquettes Graphiques Haute Fidélité

Ce dossier héberge les maquettes finales (Figma exports, PNG, etc.) de l'application tablette Android :
- Écran de veille / Scan facial / Scan QR.
- Écran de sélection des catégories de repas.
- Écrans de confirmation (succès d'enregistrement) et messages d'erreur (déjà enregistré, restaurant fermé).
- Interfaces d'administration (création d'employés, enrôlement biométrique).
