# Dossier Components — Design System & Composants Réutilisables

Ce dossier documente les spécifications de chaque composant de l'interface utilisateur de l'application :
- Boutons tactiles de repas (dimensions minimales de 48x48 dp pour le confort tactile).
- Cartes d'information, fenêtres de dialogue d'alerte, indicateurs de chargement.
- Composant caméra (cadre de centrage du visage).
