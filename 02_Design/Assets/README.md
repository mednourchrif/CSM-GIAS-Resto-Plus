# Dossier Assets — Ressources Graphiques d'Interface

Ce dossier regroupe les fichiers d'images de production, les illustrations, les icônes SVG et les polices de caractères bruts importés par l'application React Native :
- Icônes de catégories de repas (plat, pizza, sandwich).
- Icônes de statut (success, error, warning).
- Fonds d'écran, logos incorporés à l'application.
