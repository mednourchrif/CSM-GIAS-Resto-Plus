# types — Déclarations de Types TypeScript

Ce dossier regroupe les types et interfaces TypeScript partagés dans l'application mobile :
- `user.types.ts` : Structures de données pour les Employés, Stagiaires, Visiteurs.
- `meal.types.ts` : Types pour les catégories de repas (Plat, Pizza, Sandwich) et l'enregistrement de transaction.
- `api.types.ts` : Structures des réponses de l'API (succès, erreurs JWT, limitations de débit).
- `navigation.types.ts` : Typage des écrans et des paramètres de navigation React Navigation.
