# hooks — Hooks Personnalisés React Native

Ce dossier contient les hooks personnalisés (custom hooks) réutilisables à travers l'application :
- `useCamera.ts` : Gestion du cycle de vie de la caméra de la tablette, du focus et de la capture de trames.
- `useAuth.ts` : Gestion de la session administrateur JWT locale.
- `useNetwork.ts` : Surveillance de la connectivité réseau de la tablette pour gérer les micro-coupures réseau.
