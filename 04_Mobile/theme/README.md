# theme — Thème et Design System Mobiles

Ce dossier définit la charte graphique de l'application mobile React Native :
- `colors.ts` : Palette de couleurs du projet (primaires, secondaires, arrières-plans, alertes) avec ratio de contraste conforme aux exigences d'accessibilité (NFR-901).
- `spacing.ts` : Marges et paddings standardisés de l'interface.
- `typography.ts` : Familles de polices de caractères, tailles de textes et graisses autorisées.
