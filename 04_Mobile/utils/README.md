# utils — Fonctions Utilitaires Mobiles

Ce dossier héberge les fonctions d'assistance (helper functions) génériques :
- `dateHelpers.ts` : Formatage et comparaison locale des dates et heures pour le respect strict de la plage horaire 12h30 - 14h00.
- `validation.ts` : Fonctions de validation de format pour les données de création d'utilisateurs.
- `encryption.ts` : Chiffrement local basique et helpers de hachage.
