# 08_Diagrams — Fichiers Sources des Diagrammes

Ce dossier héberge les fichiers sources des modélisations visuelles du système (ex: formats `.drawio`, `.puml` pour PlantUML, ou `.svg`) :
- Modélisation de l'infrastructure réseau et physique.
- Diagrammes UML éditables.
- Logigrammes de processus d'enregistrement et d'identification.
