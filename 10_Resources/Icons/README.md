# Icons — Ressources d'Icônes Système

Ce dossier héberge les icônes vectorielles standardisées du projet pour l'interface de la tablette, le site web d'administration futur, et les documents de présentation :
- Icônes de catégories de repas (plat, pizza, sandwich).
- Icônes de statuts applicatifs (succès, échec d'authentification).
