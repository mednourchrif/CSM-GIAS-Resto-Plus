# migrations — Migrations de Base de Données (Alembic)

Ce dossier héberge les scripts d'évolution et d'historique de schéma de la base de données MySQL générés à l'aide d'Alembic (le framework de migration d'SQLAlchemy pour Python).
- Permet la traçabilité des modifications de schémas (tables, index, clés étrangères).
- Garantit la mise à niveau automatique de la base de données lors des déploiements.
