# 09_AI_Prompts — Directives d'Assistance IA pour le Projet

Ce dossier répertorie les prompts d'IA (Generative AI Engineering) optimisés et validés pour assister les développeurs lors des phases de développement et de maintenance :
- Prompts pour la génération de tests unitaires (FastAPI, React Native).
- Prompts de débogage et d'explication de code.
- Prompts d'aide à la rédaction de documentation technique additionnelle.
